# ![](assets/images/a-tracktor-2.png)

The Javascript library contains a set of widgets to visualize audio data  
and add a customizable media player to a web page.

## Source code

All the source code you can find on the
<img src="./assets/images/github.svg" width="16" />[GitHub](https://github.com/sad-systems/a-tracktor).

## Live demo

Try the [live demo](https://examples.sadspace.ru/a-tracktor)

## Installation

```
yarn add @sad-systems/a-tracktor
```
